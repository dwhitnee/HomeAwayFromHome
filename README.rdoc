== HomeAwayFromHome
================

common dotfiles and other global emacs and zsh configuration

== Installation

  git clone git://github.com/dwhitnee/HomeAwayFromHome ~/somewhere
  cd ~/somewhere
  rake install

For local overrides, you can make
 ~/.environment     # login shell overrides
 ~/.localrc         # interactive shell overrides
 ~/.emacs.local     # emacs mode overrides
